# Data

This directory contains locally cached model weights, SAE weights, and feature explanation files.
All heavy files (weights, JSONL batches) are git-ignored; only the folder structure is tracked.

| Subfolder | Contents |
|---|---|
| `deepseek-r1-distill-llama-8b/` | DeepSeek-R1-Distill-Llama-8B model weights + Llama-Scope SAE weights + Neuronpedia explanations |
| `gemma-2b/` | Gemma-2B activation dataset batches |
| `gemma-2b-it/` | Gemma-2B-IT model weights + residual-stream SAE weights |
| `qwen3-1.7b/` | Qwen3-1.7B model weights + SAE weights + Neuronpedia explanations |
