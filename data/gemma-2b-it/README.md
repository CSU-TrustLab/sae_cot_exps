# gemma-2b-it

Model weights and feature explanation JSONL files for Gemma-2B-IT with its
residual-stream SAE at layer 12 (`gemma-2b-it-res-jb`). Used in `src/experiment001.py`.

## Model weights

Downloaded via HuggingFace (`hub/`):

- **Gemma-2B-IT**: https://huggingface.co/google/gemma-2b-it
- **Gemma-2b-IT Residual Stream SAEs**: https://huggingface.co/jbloom/Gemma-2b-IT-Residual-Stream-SAEs

## Feature explanations

JSONL batch files downloaded from Neuronpedia:

https://neuronpedia-datasets.s3.us-east-1.amazonaws.com/v1/gemma-2b-it/12-res-jb/
