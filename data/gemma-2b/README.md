# gemma-2b

Feature explanation JSONL files for Gemma-2B residual-stream SAEs (`gemma-2b-res-jb`).
Used in the SAELens tutorial (`external/tutorial.py`).

## Model weights

Downloaded via HuggingFace (`hub/`):

- **Gemma-2B**: https://huggingface.co/google/gemma-2b
- **Gemma-2b Residual Stream SAEs**: https://huggingface.co/jbloom/Gemma-2b-Residual-Stream-SAEs

## Feature explanations

JSONL batch files downloaded from Neuronpedia:

https://neuronpedia-datasets.s3.us-east-1.amazonaws.com/v1/gemma-2b/

Relevant SAE layers: `0-res-jb`, `6-res-jb`
