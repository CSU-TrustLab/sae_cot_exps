# gemma-2b

Feature explanation JSONL files for Gemma-2B residual-stream SAEs (`gemma-2b-res-jb`),
downloaded from Neuronpedia. Used in the SAELens tutorial (`external/tutorial.py`).

## Source

https://neuronpedia-datasets.s3.us-east-1.amazonaws.com/v1/gemma-2b/

Relevant SAE layers: `0-res-jb`, `6-res-jb`

HuggingFace model: https://huggingface.co/google/gemma-2b  
HuggingFace SAEs: https://huggingface.co/jbloom/Gemma-2b-Residual-Stream-SAEs
